🎬 Movie Recommender System

This project uses the MovieLens dataset to recommend movies based on collaborative filtering using cosine similarity. It’s built in Python using pandas and scikit-learn.

⚙️ How It Works

# Sample Input:
Enter a movie you like:
> Toy Story (1995)

# Output:
Top 10 recommendations:
1. A Bug's Life (1998)
2. Monsters, Inc. (2001)
3. Finding Nemo (2003)
...
